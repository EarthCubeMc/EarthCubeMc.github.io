Faction

- l'explication pour le plugin de faction -
Listes des commandes de faction disponibles :

- /f create (nom de faction)Ps: une faction coute de l'or
- /f tag (nom de faction) : pour changer le name de faction
- /f join (nom de faction)
- /f promote : pour mettre vos membres en modo, ...
- /f demote : pour faire descendre en grade
- /f show ou /f who : pour voir votre faction
- /f claim : pour claim votre territoire. Ps : un claim coute de l'or
- /f unclaim : pour enlever un territoire (ps:tu regagne du gold)
- /f map : pour voir les territoires
- /f sethome : pour mettre le home de votre faction
- /f home : pour se tp a votre home de faction
- /f title : pour mettre un titre a un membre de votre faction
- /f ban : ban un membre de votre faction
- /f unban : pour révoquer le ban
- /f kick : kick un membre de votre faction
- /f list : voir toutes les factions
- /f chest : votre coffre de faction (virtuel)
- /f chat (name) : pour parler avec [Faction|Ally|Public]
- /f updates : le menu d'amélioration
- /f deposit : Pour transférer de son argent vers la banque de faction
- /sell wand : Pour transformer son or en monnaie virtuelle

Les commandes de faction indisponibles :

- /f vault est désactivé
- /f shop est désactivé
